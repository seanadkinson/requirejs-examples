The Old Way

List out all files

  • Create a huge list of scripts in a file
  • Make sure the dependency ordering is correct
  • Maybe you concanate all the scripts for production

Why is this bad?

  • Pollute the global namespace
  • Dependencies are not clearly defined (hard to test)
  • Creating smaller builds can be extremely difficult
